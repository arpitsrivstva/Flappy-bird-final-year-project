The Game


It is a 2d game which is intended for casual entertainment. We built this game with the idea of
retro games which we had in the previoius decade as video games. The game is built like that
for the fans of those genre and with that neat environment, which also gives users a feeling of
those times. It is totally free to download and you can explore this website for various things
about the game and other stuff related to it and us.

Gameplay

How-to-Play in 3 steps ~

1. Don't touch the ground! "Blocks" are the obstacles.

2. Left-clicks cause the bird to fly (kinda) or uplift a bit.

Holding the right one, helps our Flappy Bird to pass through the obstacles, but only for 5 left-clicks.
(Game-Update) - There's a "Timer" on the top right, that will make you count those left-clicks for you.

3. You have to collect the "Lights" as much as you can
to keep the game visible, yes (the brightness).

You will get it while playing, dont worry.
